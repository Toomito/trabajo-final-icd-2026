# Trabajo Final - Introducción a Ciencia de Datos 2026

Trabajo final de la materia **Introducción a Ciencia de Datos** de la 
**Universidad Católica Argentina (UCA)**, año 2026.

## Objetivo

Predecir si un cliente de tarjeta de crédito va a incumplir el pago del próximo 
mes (1) o no (0), utilizando técnicas de Machine Learning supervisado.

## Dataset

Se utiliza el dataset **"Default of Credit Card Clients"** del UCI Machine Learning 
Repository, disponible en Kaggle:

[https://www.kaggle.com/datasets/uciml/default-of-credit-card-clients-dataset](https://www.kaggle.com/datasets/uciml/default-of-credit-card-clients-dataset)

Contiene información de **30.000 clientes de tarjetas de crédito de Taiwán** 
recolectada en 2005, con 25 variables que incluyen datos demográficos, historial 
de pagos, montos facturados y montos pagados durante 6 meses.

## Tecnologías utilizadas

  - `matplotlib` y `seaborn` - visualización
